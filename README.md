# Brianna Wohner Wellness Mockup

This is a Next.js + Tailwind mockup based on the provided Brianna Wohner moodboard and mini style guide.

## Brand Direction

Primary feel:
- White editorial background
- Dark green as the main brand color instead of black
- Coral CTA buttons
- Gold highlights inspired by the crown and tropical warmth
- Teal, lime, and pink used only as small accents
- Large serif display headings
- Open Sans body copy
- Script used sparingly as a brand accent

## Colors

- Dark Green: #173D26
- Teal: #5AA6B4
- Lime: #BABC00
- Gold: #F7CB3A
- Coral: #F76F34
- Pink: #EA6C93
- Cream: #FFFDF7

## Fonts

- Noto Serif Display for headings
- Open Sans for body text
- Dancing Script used as a script stand-in for the logo/coaching accent

## How to Run

```bash
npm install
npm run dev
```

Then open:

```bash
http://localhost:3000
```
